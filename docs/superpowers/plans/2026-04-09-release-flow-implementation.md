# Release Flow Implementation Plan

> **For agentist workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Create GitHub Actions workflow that builds CleanMac, packages .dmg, generates changelog from Conventional Commits, and creates a GitHub Release.

**Architecture:** Single workflow file with sequential steps: checkout → setup-xcode → build → package → changelog → bump/tag → push → release. macOS runner required for xcodebuild and hdiutil.

**Tech Stack:** GitHub Actions, `maxim-lobanov/setup-xcode@v1`, `softprops/action-gh-release@v1`, `orhun/git-cliff-action`, `hdiutil` (built-in)

## Task 1: Create GitHub Actions Workflow

**Files:**
- Create: `.github/workflows/release.yml`

- [ ] **Step 1: Create workflow file**

```yaml
name: Release

on:
  workflow_dispatch:
    inputs:
      version_type:
        description: "Version bump type"
        required: true
        default: "patch"
        type: choice
        options:
          - patch
          - minor
          - major

jobs:
  build-and-release:
    runs-on: macos-14
    permissions:
      contents: write
      id-token: write

    steps:
      - name: Checkout repository
        uses: actions/checkout@v4
        with:
          fetch-depth: 0

      - name: Setup Xcode
        uses: maxim-lobanov/setup-xcode@v1
        with:
          xcode-version: "15.4"

      - name: Configure Git
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
        run: |
          git config --local user.email "github-actions[bot]@users.noreply.github.com"
          git config --local user.name "github-actions[bot]"

      - name: Build CleanMac
        run: |
          xcodebuild -project src/CleanMac.xcodeproj -scheme CleanMac -configuration Release build CODE_SIGN_IDENTITY="-" CODE_SIGNING_REQUIRED=NO CODE_SIGNING_ALLOWED=NO

      - name: Find built app
        id: app_path
        run: |
          APP_PATH=$(xcodebuild -project src/CleanMac.xcodeproj -scheme CleanMac -configuration Release -showBuildSettings 2>/dev/null | grep "BUILT_PRODUCTS_DIR" | awk '{print $3}')
          echo "app_path=$APP_PATH" >> $GITHUB_ENV

      - name: Package DMG
        id: dmg
        run: |
          VERSION=$(git describe --tags --abbrev=0 | sed 's/v//')
          FULL_VERSION="${VERSION}.0"
          echo "version=$FULL_VERSION" >> $GITHUB_ENV
          DMG_PATH="CleanMac-${FULL_VERSION}.dmg"
          hdiutil create -volname "CleanMac-${FULL_VERSION}" -srcfolder "${{ env.app_path }}/CleanMac.app" -ov -format UDZO "$DMG_PATH"
          echo "dmg_path=$DMG_PATH" >> $GITHUB_ENV

      - name: Install git-cliff
        run: |
          cargo install git-cliff || brew install git-cliff

      - name: Generate changelog and bump version
        id: changelog
        run: |
          git-cliff --config cliff.toml --unreleased --tag v${{ github.event.inputs.version_type }} --output CHANGELOG.md
          git-cliff --config cliff.toml --bump --remote origin
          NEW_VERSION=$(git describe --tags --abbrev=0 | sed 's/v//')
          echo "new_version=$NEW_VERSION" >> $GITHUB_ENV

      - name: Push changes and tag
        run: |
          git push origin main
          git push origin "v${{ env.new_version }}"

      - name: Create GitHub Release
        uses: softprops/action-gh-release@v1
        with:
          tag_name: v${{ env.new_version }}
          name: v${{ env.new_version }}
          body_path: CHANGELOG.md
          draft: true
          prerelease: false
          assets: ${{ env.dmg_path }}
```

## Task 2: Create git-cliff Configuration

**Files:**
- Create: `cliff.toml`

- [ ] **Step 1: Create cliff.toml**

```toml
[changelog]
header = """
# Changelog\n
"""
body = """
{% for group, commits in commits | group_by(attribute="message") %}
## {{ group | upper_first }}\n
{% for commit in commits %}
- {{ commit.message | trim | split(pat="\n") | first }}\n
{% endfor %}
{% endfor %}
"""
trim = true
footer = """
<!-- Generated by git-cliff -->
"""

[git]
tag_resolver = "semver"
conventional = true
```
